Tenho foco em boas práticas (SOLID, Clean Code, padrões GoF), documentação em Jira/Confluence e versionamento no Bitbucket/GitHub.

BasicCalc's – Calculadoras Financeiras

HTML5 • CSS3 • JavaScript • Chart.js

O BasicCalc é um projeto de desenvolvimento front-end com o objetivo de oferecer ferramentas financeiras simples e intuitivas. Construído do zero, o projeto demonstra a capacidade de criar aplicações web responsivas e dinâmicas para auxiliar o usuário no gerenciamento de suas finanças pessoais.

  • Desenvolvimento front-end com HTML, CSS.
  • Implementação da lógica de negócio para cálculo de juros e gestão de despesas com JavaScript.
  • Integração com a biblioteca Chart.js para gráficos interativos.
  • Design de uma experiência de usuário limpa e responsiva.

EasyWork – Plataforma de Serviços

React Native • Node • MongoDB

O EasyWork é um aplicativo que conecta clientes a profissionais autônomos para realização de serviços do dia a dia, como reparos domésticos, jardinagem, passeios com pets, entre outros. Inspirado no modelo “Uber de serviços”, o projeto busca simplificar a contratação e trazer mais oportunidades para trabalhadores locais.

  • Cadastro e login com autenticação via Firebase
  • Integração com banco de dados MongoDB para gerenciar clientes, profissionais e pedidos
  • Busca por serviços com listagem de profissionais disponíveis em tempo real
  • Agendamento de serviços com data, horário, local e forma de pagamento
  • Versão mobile (React Native) e versão web (React.js)
